== Description ==

A plugin that works as an example of how you can easily write your own widget. Feel free to use it as a blueprint for your own plugin. In the latest version, you can post your banner as a widget.

== Installation ==

Here are a few main steps that you need to make the plugin working:

1. Download this plugin;
2. Put the `wiget_poster_s.php` to the `/wp-content/plugins/` directory;
3. Activate the plugin through the 'Plugins' menu in WordPress;

== Frequently Asked Questions ==

= Why did you make this plugin? =
I just want to share my experience in widget making.

= It doesn't work for the latest WordPress version, what do? =
This plugin currently supports WP version up to 3.0.1, but I'm planning to release refactoring and fixes for the latest version.
